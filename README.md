### Purpose
This project is designed to automate the workflow of a network of medical laboratories. The system enables convenient management of test orders, biomaterial processing, delivery status monitoring, and storage of research results. The main goal is to increase laboratory efficiency and ensure accuracy and timeliness of diagnostics.

### Key Features
- **Order Management:** patient admission, registration, creation of test orders, identification and labeling of biomaterials.
- **Biomaterial Processing:** collection, transportation, and processing of various types of biomaterials (blood, urine, feces, sputum, saliva, cerebrospinal fluid, biopsy samples).
- **Delivery Monitoring:** creation and management of biomaterial deliveries, laboratory selection, delivery status control (pending, completed, canceled).
- **Results Storage:** storage of test results with all necessary data (patient name, order number, sample collection date and time, list of indicators, measurement units).
- **User Roles:** clear access separation based on roles (registrar-cashier, inventory manager, biomaterial transporter, etc.).
